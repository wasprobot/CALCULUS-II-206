\Que{ $ \int{\ln(\sqrt{x})}dx $ }
\Ans
{
Let $ u=\ln(\sqrt{x}) $ and $ v^{\prime}=1 $.\\

$\frac{d}{dx}\ln(\sqrt{x}) = \frac{1}{\sqrt{x}}\frac{d}{dx}x^{\frac{1}{2}} $\\

$ = \frac{1}{\sqrt{x}}\frac{1}{2}x^\frac{-1}{2} $\\

$ = \frac{1}{2}\frac{1}{\sqrt{x}}\frac{1}{\sqrt{x}} $\\

$ = \frac{1}{2x} $\\

Therefore $ u^{\prime}=\frac{1}{2x} $ and $ v=x $\\

According to integration by parts:\\
$$\int{uv^{\prime}}=uv-\int{vu^{\prime}}$$\\

Therefore $ \int{\ln(\sqrt{x})}dx $\\

$ =x\ln(\sqrt{x}) - \int{\frac{1}{2}} dx $\\

$ =x\ln(\sqrt{x}) - \frac{x}{2} + C $\\

}
