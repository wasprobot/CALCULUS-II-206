\Que{ $\int{te^{-3t}}dt$ }
\Ans
{

Let $u=t;v^{\prime}=e^{-3t}$. Therefore $v=-\frac{1}{3}e^{-3t}$\\

According to integration by parts:\\
$$\int{uv^{\prime}}=uv-\int{vu^{\prime}}$$\\

Therefore $\int{te^{-3t}}dt
= -t\frac{1}{3}e^{-3t} - \int{-\frac{1}{3}e^{-3t}}dx$\\

$= -\frac{t}{3}e^{-3t} + \frac{1}{3}\int{e^{-3t}}dx$\\

$= -\frac{t}{3}e^{-3t} - \frac{1}{9}e^{-3t} + C$\\

}
